dranges.reftuple

To Be Documented. Lots of warts still.

This module defines a function that takes lvalues and assign them, while deconstructing. 

int a, b;
_(a,b) = tuple(b,a); // swap
_(a,b) = tuple(b,a+b); // fibonacci

int[] arr = [0,1,2,3,4];
int [] c;
_(a,b,c) = arr; // a = 0, b = 1, c = [2,3,4]
